boolean && 실행문

  • (조건) && 연산자: A, B 조건 모든 참인 경우
  • 결과: true && 구문 -> && 뒤의 구문 실행
  • 결과: false && 구문 -> &&의 결과(조건)가 거짓이면 &&의 뒤를 읽지 않고 종료
    즉 실행되지 않음
